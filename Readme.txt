1.move M5 directory files into Arduino\hardware\arduino\avr\libraries\
2.In Arduino menu "Project"-->"Include Library"-->"Add .zip Library", select the M5 directory you just move
3.It will show "Library added to your libraries"
Now you can use M5 core module

4.copy the libraries directory into Arduino\
5.open the Kidsbox_gizwits.ino 
Now you can use gizwits cloud service

6.install IOE_Demo_V1.5.9_2016020616.apk on your android phone or IOE_Demo_V1.5.9_2016020616.ipa on iphone
